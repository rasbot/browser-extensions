# browser-extensions

A collection of Chrome extensions.

## Extensions

### SVEN — Swedish/English Translator

A Chrome extension that translates between English and Swedish by selecting text on any webpage.

**Features:**
- Select text on any page to see a translation tooltip
- Toggle translation direction (English → Swedish or Swedish → English) via the popup
- Google Docs support (copy text with Ctrl/Cmd+C to trigger translation)
- Uses the [MyMemory](https://mymemory.translated.net/) translation API
- 500-character selection limit

**Installation:**
1. Open `chrome://extensions` in Chrome
2. Enable "Developer mode"
3. Click "Load unpacked" and select the `chrome/sv-en-translator` directory

**Usage:**
- Select text on any webpage — a tooltip with the translation appears near the selection
- Click the extension icon to toggle between English → Swedish and Swedish → English
